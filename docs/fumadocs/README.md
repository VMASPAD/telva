# Telva Fumadocs Documentation

This directory contains a complete MDX documentation set structured for Fumadocs.

## Location

- Source root: `docs/fumadocs/content/docs`
- Navigation files: `meta.json` in each folder
- Pages: `.mdx` files with frontmatter

## Coverage

The docs include:

- end-to-end setup and quickstart
- React and Next.js integration contexts
- controlled and imperative usage patterns
- asset/export and multiplayer workflows
- `Telva` props and callbacks reference
- `TelvaApp` command cookbook
- extension patterns for React registry and custom shapes
- troubleshooting notes for rendering/export pitfalls

## Notes

- Content is written in English.
- Examples are based on the current repository implementation and example apps.
- This is additive and does not replace the existing `docs/` Markdown set.
