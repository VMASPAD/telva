# telva-core

Core rendering engine and editor utilities used by Telva.

## Install

```bash
npm i telva-core
```

## Peer dependencies

- `react >= 16.8`
- `react-dom >= 16.8`

## What it includes

- Core state and editor primitives
- Rendering and interaction layers
- Shape and geometry integration with `telva-vec` and `telva-intersect`

## Workspace scripts

```bash
yarn dev
yarn build
yarn test
yarn lint
```
